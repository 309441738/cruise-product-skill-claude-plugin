# cruise-product-skill · Claude Code 插件

通过技能获取邮轮产品信息，轻松掌握邮轮详情。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-product-skill-claude-plugin
/plugin install cruise-product-skill@cruise-product-skill-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-product-skill**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
